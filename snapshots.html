DotGNU Snapshots

For some of the DotGNU development projects, snapshots of the CVS repositories are created automatically.

  • CVS e-builds for Gentoo are avaliable here.
  • CVS snapshot debs are built daily and stored here: 
    deb http://ajmitch.dhis.org/debs ./ 
deb-src http://ajmitch.dhis.org/debs ./
    Note that currently releases and snapshots cannot be concurrently installed After adding these entries, run 
    apt-get update 
apt-get install treecc pnet
