Mailing lists

The main mailing list is the DEVELOPERS mailling list. Subscribe at http://dotgnu.org/mailman/listindo/developers.

In addition there is a number of specialized discussion mailing lists. A complete list follows.

Mailing Lists of the DotGNU project

  • The DEVELOPERS mailing list focuses on development work that leads towards the the release of DotGNU 0.1 in mid-July 2003.
  • The VISIONARIES mailing list is for discussion of more long-term projects, including the truly innovative ideas.
  • The PNET-DEVELOPERS mailing list is for developent discussions (and bugs) of DotGNU Portable.NET, including e.g. cscc, ilrun, pnetlib, and pnetC>. Topics include bugs, feature requests, interoperability testing, newbies looking for work, etc.
    • The PHPGROUPWARE-DEVELOPERS mailing list is for developent discussions of phpGroupWare.
  • The ANNOUNCE mailing list for press releases and other important announcements concerning the DotGNU system.
  • The AUTH mailing list is for discussing the DotGNU Virtual Identities system. This list has been silent for some time.
  • The ARCH for moving forward with designing the architecture of the DotGNU system and DotGNU platform. This list is for working on long-term stuff. It has been silent for some time.
  • The WEBSITE mailing list which supports the webmaster of the DotGNU website.

Please avoid cross-posting to multiple of these lists. If something needs a broader audience than just one of the specialized mailing lists, post it to DEVELOPERS.
