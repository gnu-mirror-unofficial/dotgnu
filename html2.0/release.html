DotGNU 0.6 Release Planned

A DotGNU 0.6 cd-rom image release is planned for January 2005. In the meantime, please use the download links on GetDotGNU.com.

The DotGNU 0.1 CD-ROM is now mostly of historic interest; it is still available from commercial vendors such as CheapBytes.com which sells it at http://cart.cheapbytes.com/cgi-bin/cart/0070010950.html
