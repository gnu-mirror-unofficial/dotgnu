DotGNU Portable.Net Packages for avioue systems.

Contributing

These packages are built by various part-time volunteers of the project. If you are interested in packaging DotGNU for any particular platform or distribution, please contact the DotGNU developers mailing list. The releases are usually on weekends, so volunteers have enough time to provide packages. The packages will generally be slightly delayed, please bear with us.

BitTorrent Downloads

The DotGNU 0.1 cdrom iso image as well as various packages are available as BitTorrent downloads here.

FTP and HTTP Downloads

The Portable.net 0.6.0 source tarballs are available from ftp.gnu.org and its mirrors, and also from Southern Storm.

Rpms

Mandrake Rpms of Pnet 0.6.0 have been built by Stephen Compall (s11 AT member.fsf.org), and will be posted soon.

Portable.NET 0.6.0 RedHat/Fedora (S)RPMS have been built by Phillip Compton (pcompton AT proteinmedia.com) as part of the Fedora project.

The packages are currently available at http://phillip.compton.name/pnet/. This is a temporary location until they have cleared Fedora QA.

Gentoo Ebuilds

Rainer Groesslinger (rainer.groesslinger AT gmx DOT net) has contributed ebuild rules to the Gentoo portage directory. The ebuilds are available from Gentoo to build off our gnu.org mirrors.
