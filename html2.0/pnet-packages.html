DotGNU Portable.Net Packages For Various Systems

Source Code Downloads

The Portable.net 0.6.4 source tarballs are available from ftp.gnu.org and its mirrors, and also from Southern Storm.

  • pnet-0.6.4
  • pnetlib-0.6.4
  • pnetC-0.6.4
  • ml-pnet-0.6.4
  • treecc-0.3.0
  • pnetmark-0.0.6
  • pnetcurses-0.0.2

Index of system-specific packages

Portable.Net 0.6.4 has just been released. Packages for various systems will be built over the course of the next few days.

Currently packages are available for the following systems:

GNU/Linux distributions: Gentoo (ebuilds) RedHat/Fedora (RPMs) PLD (RPMs)

Other systems: Microsoft Windows

Volunteers wanted: If there are no packages for your system yet, please consider building the missing packages and letting us know via the DotGNU developers mailing list.

Gentoo Ebuilds

Rainer Groesslinger (rainer.groesslinger AT gmx DOT net) has contributed ebuild rules to the Gentoo portage directory. Gentoo users can build and install DotGNU Portable.Net 0.6.4 by means of the following commands:

emerge sync
emerge pnet
emerge pnetlib

If you'd like to try the C compiler, you'll also want the C library:

emerge pnetc

RedHat/Fedora RPMs

Portable.NET 0.6.4 RedHat/Fedora RPMs have been built by Dejan Lekic (dejan at nu6 dot org). These packages should work on any RedHat system with a reasonably new libc.

The packages are available at http://apt.linuks.org/fedora/2.0/i386/RPMS.linuks/.

APT for RPM users can use the following: 

# linuks.org APT (RPM) repository
rpm http://apt.linuks.org fedora/2.0/i386 os linuks updates
rpm-src http://apt.linuks.org fedora/2.0/i386 os linuks updates

Packages for PLD

Thanks to Andrzej Dopierala (undefine AT aramin DOT net), PLD packages are available at ftp://ftp.pld-linux.org/dists/ac/PLD/$arch/PLD/RPMS/ where $arch is one of the following: i386, i586, i686, athlon, sparc, ppc, alpha, amd64. (For amd64, pnetC is currently not available). To install simply use poldek:

poldek --up; poldek --upa; poldek -vi pnet-compiler-csharp

Installer package for the Microsoft Windows system

Maciek Plewa (maciekplewa AT wp DOT pl) has built an installer package for the Microsoft Windows system. The package includes pnet, pnetlib, ml-pnet, pnetC, treecc, HTML documentation and samples (forms, pnetmark); it creates a new Start Menu entry with a PNet console; it's compiled with Cygwin (cygwin1.dll is included in the package).

http://download.freedom.biz/PNET-0.6.4-Setup.exe (8MB)
