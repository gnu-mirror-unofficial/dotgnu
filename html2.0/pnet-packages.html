DotGNU Portable.Net Packages For Various Systems

Under Construction

When Portable.Net 0.6.2 was released we were not able to update this website due to a problem with Savannah webcvs. Now that this problem is fixed, we're trying to hunt down what packages were built for Portable.Net 0.6.2 - sorry about the inconvenience!

Source Code Downloads

The Portable.net 0.6.2 source tarballs are available from ftp.gnu.org and its mirrors, and also from Southern Storm.

  • pnet-0.6.2
  • pnetlib-0.6.2
  • pnetC-0.6.2
  • ml-pnet-0.6.2
  • treecc-0.2.8
  • pnetmark-0.0.6
  • pnetcurses-0.0.2

Gentoo Ebuilds

Rainer Groesslinger (rainer.groesslinger AT gmx DOT net) has contributed ebuild rules to the Gentoo portage directory. Gentoo users can build and install DotGNU Portable.Net by means of the following commands:

emerge pnet
emerge pnetlib
