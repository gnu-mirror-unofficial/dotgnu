DotGNU Portable Nim

Mohan Embar wrote an implementation of the traditional "Nim" game to demonstrate how to write a Model-View-Controller application using DotGNU Portable.NET's System.Windows.Forms implementation.

dotgnu-nim.tar.gz

After unpacking the distribution, use "ilrun nim" to run the game. See the README file in the distribution for further insYou can now use DotGNU Portable.NET with our System.Windows.Forms implementation also on the Microsoft Windows System.

Please consider helping with implementing more System.Windows.Forms stuff.
