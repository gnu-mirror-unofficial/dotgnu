DotGNU Portable Nim

Mohan Embar wrote an implementation of the traditional "Nim" game to demonstrate how to write a Model-View-Controller application using DotGNU Portable.NET's System.Windows.Forms implementation.

dotgnu-nim-20040305.tar.gz

After unpacking the distribution, use "ilrun nim" to run the game. See the README file in the distribution for further information.

Note: The Nim game was originally written for Portable.Net 0.6.0, however this is an updated version which uses DrawArc (which wasn't implemented in pnetlib-0.6.0).

Please consider helping with implementing more System.Windows.Forms stuff.
