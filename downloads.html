DotGNU Downloads

The DotGNU project is a meta-project with a number of subprojects. This page serves as a one-stop spot to obtain all the available downloads for the DotGNU project. Also it provides a mirror for the source tarballs of some of the project releases. We will in the future also list other free-software projects supported by DotGNU.

DotGNU Portable.Net 0.5.4

The Portable.net 0.5.4 tarballs are available from ftp.gnu.org and from Southern Storm. GnuPG-signed md5sums are at the end of the release announcement.

Source

  • pnet-0.5.4
  • pnetlib-0.5.4
  • treecc-0.2.4
  • pnetmark-0.0.4
  • pnetcurses-0.0.2

Debs

The location of the binaries is ftp.gnu.org. Use these for apt-get support. 

deb ftp://ftp.gnu.org/gnu/dotgnu-pnet/debian ./ 
deb-src ftp://ftp.gnu.org/gnu/dotgnu-pnet/debian ./

Rpms

Rpms for Mandrake 9.0 have been built by Stephen Compall (s11 AT member.fsf.org) who remarks: "Please note that there are a couple of new packages: pnet-libgc and pnet-embed-devel. I added these in a hurry for a place to put some newly installed stuff; they aren't really useful for most purposes. But they're there if you need them."

Source RPMS for Redhat have been built by Phillip Compton ( pcompton AT proteinmedia.com) as part of the Fedora project.

  • pnet-0.5.4-0.fdr.1.src.rpm
  • pnetlib-0.5.4-0.fdr.1.src.rpm
  • treecc-0.2.4-0.fdr.1.src.rpm

SRPMs are also available.

Binaries for the Microsoft Windows system

Robert Johnson (bobjohnson11 AT yahoo DOT com) has built the Portable.net system and the treecc tool for the Microsoft Windows system. They are available (with an installer) as zip packages.

  • Portable.net 0.5.4 package
  • TreeCC 0.2.4 package

Mac OS X Packages

The Mac .pkg files will soon become available as soon as they are tested.

Phpgroupware 0.9.14

The Phpgroupware team has been working on their admirable webservices package and have rolled out their 0.9.14.002 release. They have a lot of packages, so visit their download section. Phpgroupware is part of the DotGNU webservices project and supports major webservice protocols.

DotGNU Execution Environment 0.1.4

The DGEE is an appserver which currently uses the Portable.Net runtime to execute the services. The DGEE is centered around its own Savannah project . DGEE uses Apache to manage HTTP traffic for its xmlrpc frontends. GnuPG-signed md5sums are here.

Source

  • phlib-1.18.tgz
  • goldwater-0.3.4.tgz
  • dgee-0.1.4.tgz
