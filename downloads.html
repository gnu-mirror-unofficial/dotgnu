DotGNU Downloads

The DotGNU project is a meta-project with a number of subprojects. This page serves as a one-stop spot to obtain all the available downloads for the DotGNU project. Also it provides a mirror for the source tarballs of some of the project releases. We will in the future also list other free-software projects supported by DotGNU.

DotGNU Execution Environment 0.1.2

The DGEE is an appserver which currently uses the Portable.Net runtime to execute the services. The DGEE is centered around its own Savannah project . DGEE uses Apache to manage HTTP traffic for its xmlrpc frontends.
