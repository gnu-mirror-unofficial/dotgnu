DotGNU Downloads

The DotGNU project is a meta-project with a number of subprojects. This page serves as a one-stop spot to obtain all the available downloads for the DotGNU project. Also it provides a mirror for the source tarballs of some of the project releases. We will in the future also list other free-software projects supported by DotGNU.

DotGNU Portable.Net 0.5.2

The Portable.net 0.5.2 tarballs are available from Southern Storm. These have been mirrored on this server.

Source

  • pnet-0.5.2
  • pnetlib-0.5.2
  • pnetC-0.5.0 (no new release)
  • treecc-0.2.2
  • pnetmark-0.0.4
  • pnetcurses-0.0.2

Debs

The location of the binaries is ftp.gnu.org. Use these for apt-get support. 

deb ftp://ftp.gnu.org/gnu/dotgnu-pnet/debian ./ 
deb-src ftp://ftp.gnu.org/gnu/dotgnu-pnet/debian ./

Microsoft Windows binaries

Robert Johnson (bobjohnson11 AT yahoo DOT com) has built the Portable.net system and the treecc tool for Windows systems. They are available as zip packages.

  • Portable.net 0.5.2 package
  • TreeCC 0.2.2 package

Mac OS X Packages

The Mac .pkg files will soon become available as soon as they are tested.

Phpgroupware 0.9.14

The Phpgroupware team has been working on their admirable webservices package and have rolled out their 0.9.14 release. They have a lot of packages, so visit their download section. Phpgroupware is part of the DotGNU webservices project and supports major webservice protocols.

DotGNU Execution Environment 0.1.2

The DGEE is an appserver which currently uses the Portable.Net runtime to execute the services. The DGEE is centered around its own Savannah project . DGEE uses Apache to manage HTTP traffic for its xmlrpc frontends.

Source

  • phlib-1.17.tgz
  • goldwater-0.3.2.tgz
  • dgee-0.1.2.tgz
