DotGNU Downloads

The DotGNU project is a meta-project with a number of subprojects. This page serves as a one-stop spot to obtain all the available downloads for the DotGNU project. Also it provides a mirror for the source tarballs of some of the project releases. We will in the future also list other free-software projects supported by DotGNU.

Contributing

These packages are built by various part-time volunteers of the project. If you are interested in packaging DotGNU for any particular platform or distribution ,please contact the DotGNU developers mailing list. The releases are usually on weekends , so volunteers have enough time to provide packages . The packages will generally be slightly delayed, please bear with us.

Phpgroupware

PhpGroupWare is available for download from the download section on phpgroupware.org. Phpgroupware is part of the DotGNU webservices project and supports major webservice protocols.

DotGNU Execution Environment 0.1.4

The DGEE is an appserver which currently uses the Portable.Net runtime to execute the services. The DGEE is centered around its own Savannah project . DGEE uses Apache to manage HTTP traffic for its xmlrpc frontends. GnuPG-signed md5sums are here.
