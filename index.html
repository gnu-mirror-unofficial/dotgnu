DotGNU - Making the Web Work for You!

We're building something that could be called an "operating system for the internet" - a collection of tools that allow software components to interact with each other over the internet. The most impressive parts are currently DotGNU Portable.NET which builds a C# compiler, class library, and related development tools, and phpGroupWare, a multi-user web-based GroupWare suite, which also serves to provide a good collection of webservice components, all of which can be accessed through XML-RPC so that you can easily integrate them into webservice applications of your own.

DotGNU, a project of GNU, is really a meta-project that will consist of a good number of such software development projects. The main objectives of DotGNU are to build a webservices platform and a Virtual Identities system.

News

Oct 28-10-2002: Portable.NET's cscc compiles pnetlib

With the application of a small patch to pnetlib (it changes only 5 lines), cscc can compile all of pnetlib. This means pnetlib hackers can directly test their modifications.

Oct 26-10-2002: Weekly IRC Meeting

The Agenda for the weekly IRC meeting has been announced here. It includes support for Floating point support, PMCodeGen implementation, new language plugins for cscc and the type masking bug in cscc and other issues.

Oct 25-10-2002: Daily builds of pnetlib

Charles Schuller (Kyeran) has provided us with a daily build of Portable.Net libraries. A mirror of these as well as tarballs of CVS HEAD source has been provided by Stephen Compall (S11001001).

Oct 19-10-2002: IRC Meeting with Parrot Developers

The IRC meeting to discuss adding Parrot to the bytecodes supported by Portable.Net was a great success. The Parrot designer was here to discuss the details of generating Parrot from C#. Read all about it here.

Oct 19-10-2002: IRC Meeting Agenda

The Agenda for the weekly IRC meeting has been announced here. It includes support for Parrot , Using Treecc for wrapping gcc and other issues.

Oct 18-10-2002: DotGNU Focus and Direction

The DotGNU project is undergoing a restructuring due to the reduced threat of Microsoft's .NET deployment plan. The plans for a new and improved version of DotGNU can be found here. here. The focus shift is from total .NET compatibility to a more useful and creative applications of DotGNU technologies.

Oct 12-10-2002: Weekend Warriors Programme

To speed up the compiler development , Peter Minten, Gopal.V and Andrew Mitchell have formed the Weekend Warriors. This is a group of part time developers who devote their weekends to hacking pnet. This is responsible for the flurry of activity which happens on every weekend. The group proposes to take part of the compiler development in the following announcement.: Weekend Warriors

Oct 10-10-2002: Meetathon - III

As is customary, DotGNU will be having a Meet-a-thon this quarter as well. This is an event to take stock of the last quarter for the DotGNU developers, and an excellent oppurtunity for newcomers to familiarize themselves with the project. See the proposal for this

Oct 02-10-2002: Java Bytecode Output for Portable.Net

Sylvain Pasche has fixed up the Portable.Net assembler to generate the correct java bytecode. Now the C# compiler sports a nice JVM output backend. The results become really interesting when the whole pnetlib can be compiled into JVM code. Read full details here.
