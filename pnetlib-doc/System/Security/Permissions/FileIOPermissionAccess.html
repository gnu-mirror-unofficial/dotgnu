FileIOPermissionAccess.PathDiscovery Field

PathDiscovery = 0x8;

Summary

Specifies access to the path information for a file or directory. System.Security.Permissions.FileIOPermissionAccess.PathDiscovery does not include access to the contents of a file or directory.

[Note: This permission is used to protect sensitive information in the path, such as user names, as well as information about the directory structure revealed in the path. ]
