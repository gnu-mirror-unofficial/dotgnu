Summary

Constructs and initializes a new instance of the PathTooLongException class.

Description

This constructor initializes the System.IO.PathTooLongException.Message property of the new instance to a system-supplied message that describes the error, such as "The supplied path is too long." This message takes into account the current system culture.

The System.IO.PathTooLongException.InnerException property of the new instance is initialized to null .
