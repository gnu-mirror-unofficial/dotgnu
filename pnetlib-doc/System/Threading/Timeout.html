Summary

A constant used to specify an infinite waiting period.

Description

The value of this constant is -1.

[Note: For threading methods that accept a timeout parameter, such as System.Threading.Thread.Sleep(System.Int32) and System.Threading.Threading.Join , this value is used to suppress method behavior that would normally occur due to elapsed time. ]
