DotGNU Portable.Net Packages For Various Systems

Source Code Downloads

The Portable.net 0.6.6 source tarballs are available from ftp.gnu.org and its mirrors, and also from Southern Storm.

  • pnet-0.6.6
  • pnetlib-0.6.6
  • pnetC-0.6.6
  • ml-pnet-0.6.6
  • treecc-0.3.0
  • pnetmark-0.0.6
  • pnetcurses-0.0.2

Index of system-specific packages

Currently packages are available for the following systems:

GNU/Linux distributions: Gentoo (ebuilds), RedHat/Fedora (RPMs), Debian PLD (RPMs)

Other systems: Mac OS X, Microsoft Windows

Volunteers wanted: If there are no packages for your system yet, please consider building the missing packages and letting us know via the DotGNU developers mailing list.

Gentoo Ebuilds

Rainer Groesslinger (rainer.groesslinger AT gmx DOT net) has contributed ebuild rules to the Gentoo portage directory. Gentoo users can build and install DotGNU Portable.Net 0.6.6 by means of the following commands:

emerge sync
emerge pnet
emerge pnetlib
emerge ml-pnet

If you'd like to try the C compiler, you'll also want the C library:

emerge pnetc

The ebuilds are currently available for the following processor types: x86, ppc, sparc, mips, alpha, hppa, amd64 and ia64

RedHat/Fedora RPMs

Portable.NET 0.6.6 RedHat/Fedora RPMs have been built by Dejan Lekic (dejan at nu6 dot org). These packages should work on any RedHat system with a reasonably new libc.

The packages are available at apt.linuks.org, as follows: pnet-compiler pnet-interpreter pnet-libgc pnet-tools pnetlib ml-pnet pnet-embed-devel pnetC treecc.

Debian

Debian packages have been built by Andrew Mitchell (ajmitch at gnu dot org).

Users of Debian "unstable" can install them by making sure that in the file /etc/apt/sources.lists there is a line 

	deb http://mentors.debian.net/debian unstable main

and then: 

	apt-get install pnet
	apt-get install ml-pnet      # additional libs from mono (optional)

Packages for PLD

Thanks to Andrzej Dopierala (undefine AT aramin DOT net), PLD packages are available at ftp://ftp.pld-linux.org/dists/ac/PLD/$arch/PLD/RPMS/ where $arch is one of the following: i386, i586, i686, athlon, sparc, ppc, alpha, amd64. (For amd64, pnetC is currently not available). To install simply use poldek:

poldek --up; poldek --upa; poldek -vi pnet-compiler-csharp

Mac OS X

Portfiles for the DotGNU Portable.Net 0.6.6 components (pnet, pnetlib, ml-pnet, treecc) are available through the Darwinports system.

Installer package for the Microsoft Windows system

Maciek Plewa (maciekplewa AT wp DOT pl) has built an installer package for the Microsoft Windows system. The package is available from the milsim site.
